サードパーティの JavaScript

サードパーティの JavaScript とは、サードパーティのソースから追​​加されたスクリプトを指します。通常、分析、広告、カスタマー サポート ウィジェットなど、ゼロから作成する必要のない新しい機能をサイトに導入するために、サードパーティのスクリプトが含まれています。

サードパーティの JavaScript を追加する

サードパーティのスクリプトを Next.js ページに追加する方法を詳しく見ていきましょう。

pages/posts/first-post.jsエディターで開き、次の行を見つけます。

<Head>
  <title>First Post</title>
</Head>

メタデータに加えて、できるだけ早くロードして実行する必要があるスクリプトは通常、<head>ページ内に追加されます。通常の HTML<script>要素を使用すると、次のように外部スクリプトが追加されます。

<Head>
  <title>First Post</title>
  <script src="https://connect.facebook.net/en_US/sdk.js" />
</Head>

このスクリプトには、 Facebook ソーシャル プラグインやその他の機能を導入するために一般的に使用されるFacebook SDK が含まれています。このアプローチは機能しますが、この方法でスクリプトを含めると、同じページでフェッチされた他の JavaScript コードに関していつロードされるかについて明確なアイデアが得られません。特定のスクリプトがレンダリングをブロックし、ページ コンテンツの読み込みを遅らせる可能性がある場合、これはパフォーマンスに大きな影響を与える可能性があります。

スクリプト コンポーネントの使用

next/scriptHTML<script>要素の拡張であり、追加のスクリプトがフェッチされて実行されるときに最適化されます。

同じファイルで、ファイルの先頭にScriptfromのインポートを追加します。next/script

import Script from 'next/script';

次に、FirstPostコンポーネントを更新してコンポーネントを含めますScript。

export default function FirstPost() {
  return (
    <>
      <Head>
        <title>First Post</title>
      </Head>
      <Script
        src="https://connect.facebook.net/en_US/sdk.js"
        strategy="lazyOnload"
        onLoad={() =>
          console.log(`script loaded correctly, window.FB has been populated`)
        }
      />
      <h1>First Post</h1>
      <h2>
        <Link href="/">← Back to home</Link>
      </h2>
    </>
  );
}

スクリプト コンポーネントでいくつかの追加プロパティが定義されていることに注意してください。

  • strategyサードパーティのスクリプトをいつロードするかを制御します。の値は、lazyOnloadブラウザーのアイドル時間中にこの特定のスクリプトを遅延してロードするように Next.js に指示します。
  • onLoadスクリプトの読み込みが完了した直後に JavaScript コードを実行するために使用されます。この例では、スクリプトが正しくロードされたことを示すメッセージをコンソールに記録します。

http://localhost:3000/posts/first-post にアクセスしてみてください。ブラウザーの開発者ツールを使用すると、上記のメッセージがConsole パネルに記録されているはずです。さらに、window.FBスクリプトを実行して、このグローバル変数が設定されたことを確認できます。

注: Facebook SDK は、パフォーマンスの高い方法でサードパーティのスクリプトをアプリケーションに追加する方法を示すための不自然な例としてのみ使用されました。Next.js にサードパーティの機能を含める基本を理解したので、先に進む前にスクリプト コンポーネントを削除できますFirstPost。

コンポーネントの詳細については、ドキュメントをScriptご覧ください。
